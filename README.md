# Assets

Central repository for reusable visual resources across the organization.  
It stores and maintains brand-related materials used in our projects and communications.

## 📦 What’s inside

- **🎨 Logos & Isologos**  
  Official logo variations, isologos, and approved brand marks in multiple formats.

- **🧩 Design Assets**  
  Icons, illustrations, banners, and other shared graphical elements intended for reuse.

- **📐 Brand Resources**  
  Foundational visual components to ensure consistency across repositories, websites, and documentation.

## 🤝 Contributing

We welcome contributions from designers interested in improving or expanding our visual identity.  
If you would like to collaborate on logos or branding assets, feel free to open an issue or submit a proposal.
