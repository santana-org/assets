# Assets

Central repository for reusable visual resources across the organization.  
It stores and maintains brand-related materials used in our projects and communications.

## 📦 What's inside

```
source/
├── logo/          ✅ Available
│   ├── main-dark.webp
│   ├── main-gradient.webp
│   ├── main-light.webp
│   └── main-solid.webp
├── favicon/       ⚠️  Pending — see source/favicon/README.md
├── icons/         ⚠️  Pending — see source/icons/README.md
└── social/        ⚠️  Pending — see source/social/README.md
```

### 🎨 Logos & Isologos
Official logo variations and approved brand marks.  
Currently available in **WebP** format with four variants: `dark`, `gradient`, `light`, and `solid`.

**Missing formats:** SVG and PNG versions of all logo variants.

### 🔖 Favicon
Browser tab icons, Apple touch icons, Android icons, and a web app manifest.  
See [`source/favicon/README.md`](source/favicon/README.md) for the full list of expected files.

### 🧩 Icons
Standalone isotype (symbol-only) assets in SVG and PNG for use where the full logo is not appropriate.  
See [`source/icons/README.md`](source/icons/README.md) for the full list of expected files.

### 📣 Social Media Assets
Open Graph images, Twitter cards, GitHub social previews, and banners for social profiles.  
See [`source/social/README.md`](source/social/README.md) for the full list of expected files.

## 🤝 Contributing

We welcome contributions from designers interested in improving or expanding our visual identity.  
If you would like to collaborate on logos or branding assets, feel free to open an issue or submit a proposal.
