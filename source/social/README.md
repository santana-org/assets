# Social Media Assets

This directory is intended to hold social media and open graph assets for the Santana brand.

## Expected files

| File | Dimensions | Purpose |
|------|-----------|---------|
| `og-image.png` | 1200×630 | Open Graph / Facebook / LinkedIn preview image |
| `twitter-card.png` | 1200×628 | Twitter (X) summary card image |
| `github-social-preview.png` | 1280×640 | GitHub repository social preview |
| `banner-light.png` | 1500×500 | Light-background banner for social profiles |
| `banner-dark.png` | 1500×500 | Dark-background banner for social profiles |
