# Logo

This directory contains the official Santana logo in its main variants.

## Available files

| File | Background | Style |
|------|-----------|-------|
| `main-dark.webp` | Dark | Standard |
| `main-gradient.webp` | Any | Gradient |
| `main-light.webp` | Light | Standard |
| `main-solid.webp` | Any | Solid monochrome |

## Missing formats

The following formats are not yet available but are needed for full brand consistency:

| Format | Purpose |
|--------|---------|
| SVG | Scalable vector — required for print, high-DPI screens, and resizing without quality loss |
| PNG | Raster with transparency — broad compatibility for web and document use |
