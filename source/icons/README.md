# Icons

This directory is intended to hold standalone icon assets for the Santana brand.

## Expected files

| File | Format | Purpose |
|------|--------|---------|
| `icon-solid.svg` | SVG | Monochrome solid icon (no wordmark) |
| `icon-gradient.svg` | SVG | Gradient icon (no wordmark) |
| `icon-solid.png` | PNG | Raster version of the solid icon |
| `icon-gradient.png` | PNG | Raster version of the gradient icon |

Icons represent the isotype (symbol-only mark) of the Santana brand and are used when space is limited or when the full logo is not appropriate.
