# Favicon

This directory is intended to hold all favicon assets for the Santana brand.

## Expected files

| File | Size | Purpose |
|------|------|---------|
| `favicon.ico` | 16×16, 32×32, 48×48 (multi-size) | Standard browser favicon |
| `favicon-16x16.png` | 16×16 | Small browser tab icon |
| `favicon-32x32.png` | 32×32 | Standard browser tab icon |
| `apple-touch-icon.png` | 180×180 | iOS home screen icon |
| `android-chrome-192x192.png` | 192×192 | Android home screen icon |
| `android-chrome-512x512.png` | 512×512 | Android splash screen icon |
| `site.webmanifest` | — | Web app manifest referencing the above icons |
